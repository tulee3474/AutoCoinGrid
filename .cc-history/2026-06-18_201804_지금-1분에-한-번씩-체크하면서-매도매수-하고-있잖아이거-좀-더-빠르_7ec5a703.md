## EC2 실제 배포 순서

```bash
# 1. EC2 인스턴스 생성 후 SSH 접속
ssh -i your-key.pem ubuntu@<EC2-PUBLIC-IP>

# 2. 초기 세팅 (최초 1회)
curl -o ec2-setup.sh https://raw.githubusercontent.com/.../deploy/ec2-setup.sh
sudo bash ec2-setup.sh

# 3. 코드 클론
git clone <your-repo> /home/ubuntu/autocoin
cd /home/ubuntu/autocoin

# 4. 환경변수 설정
cp backend/.env.example backend/.env
nano backend/.env  # DATABASE_URL(RDS), JWT_SECRET, ENCRYPTION_KEY 등 입력

# 5. 첫 배포
./deploy/deploy.sh
```

**RDS 설정 시 주의:**
- `DATABASE_URL=mysql://autocoin:password@<RDS-ENDPOINT>:3306/autocoin`
- EC2 보안 그룹에서 RDS 포트(3306) 인바운드 허용
- `CORS_ORIGIN=http://<EC2-PUBLIC-IP>` (도메인 연결 후 `https://yourdomain.com`)
